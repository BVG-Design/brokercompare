# **App Name**: BrokerCompare AU

## Core Features:

- Service Directory: Comprehensive directory of marketing, virtual assistant, and commercial finance providers in Australia, with detailed profiles.
- Software Listings: Listings of software used by brokers, including features, pricing, and compatibility information.
- Comparison Tool: Compare brokers and software side-by-side to identify the best options.
- AI-Powered Recommendation Tool: Based on the broker's and software's profile characteristics, a large language model tool recommends ideal service/software matches.
- User Reviews and Ratings: Allow users to submit reviews and ratings for both brokers and software to allow people to make an informed decision. Rate service providers, leave written comments and interact with existing posts.
- Search and Filtering: Robust search and filtering options to narrow down brokers and software based on specific criteria (e.g., location, specialization, pricing).

## Style Guidelines:

- Primary color: Dark Blue (#142848) to evoke trust, stability, and professionalism in the financial sector.
- Background color: Light Cream (#FFFFF0) for a clean and modern interface.
- Secondary color: Soft Orange (#EF4E22) to highlight key features and CTAs, providing a fresh contrast to the primary blue.
- Accent color: Fresh Green (#1BCBA7) to highlight key features and bring attention
- Body and headline font: 'Inter', a sans-serif typeface, to create a neutral and very legible feel, and improve readability of data.
- Use consistent and professional icons, primarily line icons, to represent various categories, filters, and features, improving usability.
- Implement a clean and well-organized layout, using cards or blocks to present information clearly. Ensure responsiveness across devices.
- Use subtle animations and transitions for interactive elements, like filter toggles or expanding content sections, to enhance user engagement.
